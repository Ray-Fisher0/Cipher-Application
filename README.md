Overview

This Cipher Application provides an interactive way to encode and decode text using multiple encryption techniques.
It was designed to demonstrate understanding of:
  Text manipulation
  Algorithm design
  Optional file input/output for automation

Features

Encrypt and decrypt messages with selected cipher methods
Choose between manual input or text files
Handle both uppercase and lowercase letters
Optional input/output redirection
Clear user interface for command-line operation

Supported Ciphers:

  Caesar Cipher with random amount of shifting
  More coming soon
